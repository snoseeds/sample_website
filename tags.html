Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Document Tags

Tag Name Purpose
html HTML wraps contents for HTML markup
head head section for Webpage title, scripts, and other meta-information not shown in webpages
body body contains all the visible contents of websites
title title to specify the title of a webpage
meta meta for specifying info that helps with better rendering or treatment of webpage contents

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header wraps heading sections
div div wraps contents into sections
blockquote blockquote formats a block of quote with indentation
br break makes content after it to start on the next line

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>at least a billion</strong> at least a billion
a anchor hyperlink <a href="http:learnenough.com">Learn Enough</a> Learn Enough
img image add image <img src="images/small_twitter_logo.png" alt="Learn Enough Twitter Logo">
code code formats as monospace <code><em>technical sophistication</em></code> <em>technical sophistication</em>
span span wraps contents for styling <span>Call me Ishmael.</span> Call me Ishmael.
ol ordered list make ordered list <ol>

<li>Vengeful whale</li>

<li>Salty sailors</li>

</ol>
  1. Vengeful whale
  2. Salty sailors
ul unordered list make unordered list <ul>

<li>The story pretty much happened in real life</li>

<li>Mad sea captains are fun</li>

</ul>
  • The story pretty much happened in real life
  • Mad sea captains are fun
